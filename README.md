# CQL TEACHER

Es la parte del cliente

## Getting Started

+ Clonar este repositorio
+ Instalar express 
```
npm install --save express
```
+ Iniciar Servidor
```
npm index.js
```


### Prerequisites

+ NodeJS


## Built With

* [NodeJS](https://nodejs.org/es/) - Servidor de Aplicacion para Cliente
